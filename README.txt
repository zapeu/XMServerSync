# XMServerSync

A Forge/NeoForge mod for Minecraft 1.20.1 that automatically syncs Xaero's Minimap and Worldmap data between client and server.

## Features (planned)
- Automatic sync on join/leave
- Zero manual setup required for players
- Secure data handling
- Lightweight and reliable

## Status
Early development. Not functional yet.
